🌟 프로젝트 개요: 공부의 숲 🌟

📚 목표: 습관 형성 및 집중 시간 관리를 지원하여 사용자의 꾸준한 학습을 돕는 웹 서비스 개발

🗓️ 기간: 2025년 3월 21일 ~ 2025년 4월 10일

🧑‍🤝‍🧑 팀원: 이지수B (팀장), 김홍섭, 이태빈, 안세빈, 임정빈, 최민경, 박민규

🤝 팀프로젝트 협업 Notion: 팀프로젝트 "공부의 숲" 노션 바로가기

🎬 웹사이트 바로가기: https://6-studyforest-phytoncide-fs.netlify.app/


🛠️ 주요 기술 스택

  • Frontend: HTML, CSS, JavaScript, React, React Router, Vite
  • Backend: Node.js, Express.js, PostgreSQL, Prisma
  • Deploy: Netlify
  • Tools: Git, GitHub, npm, Nodemon

✨ 주요 기능

  • 습관 관리: 매일 반복할 습관 설정 및 체크 기능
  • 집중 타이머: 학습 시간 설정 및 집중 시작/완료 기능, 집중 시간에 따른 포인트 지급
  • 스터디 그룹: 스터디 생성, 조회, 상세 정보 확인 및 수정/삭제
  • 사용자 인터랙션: 응원 이모지 추가/삭제, 최근 본 스터디 표시
  • UX 개선: 로딩 바 적용, Hover/Touch 시 인터랙션 효과
  • 검색 및 정렬: 스터디 목록 검색 (키워드), 포인트/최신순 정렬, 페이지네이션

✅ 성취 및 배운 점

  • React 기반의 사용자 인터페이스 개발 능력 향상: 컴포넌트 기반 설계, 상태 관리, 라우팅 학습
  • Node.js와 Express.js를 활용한 백엔드 개발 이해: API 설계 및 구현, 데이터베이스 연동
  • Prisma ORM을 통한 효율적인 데이터 관리: 스키마 정의, 데이터 CRUD 작업
  • 반응형 웹 디자인 구현: 다양한 해상도에 최적화된 사용자 경험 제공
  • 팀 협업 및 프로젝트 관리 능력 향상: Git/GitHub 활용, Notion을 통한 문서 공유 및 협업
  • UX/UI 개선을 위한 고민: 사용자 편의성을 고려한 기능 추가 및 인터랙션 적용
  • 데이터베이스 성능 최적화 경험: Prisma Raw Query를 활용한 특정 검색 기능 개선

🚧 기술적 도전 및 극복

  • Prisma Raw Query를 이용한 검색 기능 구현: 기존 ORM 방식의 한계를 극복하고 SQL 쿼리 수준에서 검색 로직을 구현하여 성능 향상 및 검색 정확도 개선
  • 공통 컴포넌트 및 로직 재사용성 확보: 비밀번호 검증 모달 등 공통으로 사용되는 UI 요소 및 로직을 효율적으로 관리하고 재사용하여 개발 효율성 증대
  • 프론트엔드-백엔드 데이터 연동 과정에서의 이해도 증진: API 호출 및 응답 처리, 데이터 유효성 검사 등 통합적인 개발 프로세스 경험
